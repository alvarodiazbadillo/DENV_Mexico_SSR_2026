# DENV_Mexico_SSR_2026

## Overview

This repository contains the analytical workflow, figures, supplementary materials, and documentation associated with the study:

**"Microsatellite Signatures Reveal Serotype-Specific Genomic Architecture and Evolutionary Structure in Dengue Virus Genomes: A Pilot Whole-Genome Analysis of Mexican Isolates"**

This project investigates the distribution, density, and composition of simple sequence repeats (SSRs) across complete Dengue virus genomes representing the four recognized serotypes (DENV1–DENV4).

---

## Objectives

* Characterize genome-wide SSR distribution.
* Compare SSR repertoires among DENV serotypes.
* Identify candidate serotype-specific SSR signatures.
* Explore the relationship between SSR architecture and genomic evolution.
* Evaluate SSR-based clustering and classification approaches.

---

## Dataset

This pilot study analyzed twelve complete Dengue virus genomes:

* DENV1 (n=3)
* DENV2 (n=3)
* DENV3 (n=3)
* DENV4 (n=3)

Reference genomes were included for comparative purposes.

Sequence data were obtained through GISAID under its Terms of Use.

---

## Analytical Workflow

1. Genome acquisition
2. Quality control
3. Multiple sequence alignment
4. Gap removal
5. SSR detection
6. Genomic annotation
7. SSR density analysis
8. Mutation hotspot comparison
9. Principal Component Analysis (PCA)
10. Hierarchical clustering
11. Candidate biomarker identification
